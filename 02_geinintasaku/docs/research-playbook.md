# 芸人探索MAP 調査プレイブック

このアプリの本丸は、芸人の現在地ではなく、公開発言として残っている「よく行く店」「通った店」「劇場帰りに来ていた店」を地図で楽しむこと。

## 優先順位

1. 行きつけ系
   - よく行く
   - 行きつけ
   - 通っている
   - 劇場帰りに来ていた
   - 仕事終わりに行く

2. 通った店・下積みの店
   - 若手時代に通った
   - 下積み時代を支えた
   - 大阪時代によく来ていた
   - コンビで通った

3. 思い出の店
   - 人生最高の一品
   - 家族や先輩との思い出
   - 芸人人生の節目に出てくる店

4. 番組訪問
   - 番組で実際に訪れた店
   - 街歩きロケで食べた店
   - 地図体験には向くが、会えるコンセプトでは補助扱い

5. 起点
   - 劇場
   - テレビ局
   - トークライブ会場
   - ここから周辺の行きつけ店を探す


## 採用条件

採用しやすいもの:

- 芸人本人の発言が確認できる
- 番組名、媒体名、配信回がわかる
- 店名が特定できる
- 出典URLがある
- 現在地や生活圏ではなく、過去の公開情報として扱える

採用しないもの:

- 目撃情報だけ
- ファン投稿だけ
- 自宅や生活圏の推測
- 「今いる」「今日来た」系
- サインがあるだけで発言がないもの
- 店舗側の「芸能人御用達」だけ

## ラベル運用

- 行きつけ系: 最優先。強いピン。
- 発言あり: 番組や記事で本人・番組文脈が確認できる。
- 番組訪問: 地図体験の補助。行きつけより目立たせない。
- 起点確認: 劇場やテレビ局。
- 発言候補: 後で出典確認する棚。
- 要確認メモ: 店名や番組回が未確定。
