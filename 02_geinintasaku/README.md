# 芸人に会える場所MAP

「芸人に会える場所MAP」は、芸人の現在地を追跡するアプリではありません。公開情報をもとに、芸人が話した・訪れた・紹介した場所を地図上で楽しむためのWebアプリです。

## アプリ概要

スマホで開くと、新宿周辺を中心に「芸人ゆかり」の劇場、飲食店、街スポットを地図で楽しめます。ピンを押すと店舗情報ではなく、キャッチコピー、へぇ度、関連芸人、芸人ネタ、出典メモを主役にしたカードが下から表示されます。

## コンセプト

新宿駅、ルミネtheよしもと、新宿三丁目、歌舞伎町、思い出横丁あたりを中心に、芸人カルチャーを感じる街歩き体験を作るMVPです。UIはGoogleマップに近い素直な操作感にし、独自性は「芸人ネタカード」に寄せています。

## 技術構成

- Next.js
- React
- TypeScript
- Tailwind CSS
- MapLibre GL JS
- GeoJSON / JSONによる静的データ読み込み
- DB、認証、サーバーサイド処理なし

## 起動方法

```bash
npm install
npm run dev
```

ブラウザで `http://localhost:3000` を開きます。

型チェックは以下で確認できます。

```bash
npm run typecheck
```

## データ構造

- `public/data/places.geojson`: ピンとして表示するスポット情報
- `public/data/areas.json`: エリアタブの中心座標とズーム
- `public/data/heat.geojson`: 新宿周辺の「会えそうエリア」ポリゴン

`places.geojson` の各Featureには、場所名、カテゴリ、へぇ度、関連芸人、芸人ネタ、出典メモ、タグ、GoogleマップURLを持たせています。

## 調査ルール

- AIは候補を集めるだけ
- 出典が確認できるまで断定しない
- 第三者の目撃情報だけでは採用しない
- 現在地推測は扱わない
- 自宅・生活圏推測は扱わない
- 事実情報と編集文を分けて扱う

## 注意事項

このMVPのスポットデータはダミーを含むサンプルです。芸人の現在地表示、目撃情報投稿、自宅や生活圏に関する情報、追跡や通知、管理画面は実装していません。
