# 岡山市中心市街地 GIS分析まとめ
**作成日：2026-06-24**

---

## 1. やったこと

### 地図の構築（build_overlay_map.py）

岡山市中心部の22エリアを対象に、Foliumベースのインタラクティブ地図を構築した。レイヤーは計14種類。

| レイヤー | データソース |
|----------|-------------|
| 人流圏（22エリア） | 独自調査データ（zinryu_okayama） |
| 飲食店・カフェ | Overpass API (OSM) |
| バー・居酒屋 | Overpass API (OSM) |
| スーパー | Overpass API (OSM) |
| オフィスビル | Overpass API (OSM) |
| 大規模SC | Overpass API (OSM) |
| バス停・便数 | 岡山市GTFSデータ |
| 緑地・公園 | Overpass API (OSM) |
| 駐車場 | PLATEAU CityGML + OSM |
| 夜間人口 | 独自データ |

各エリアには人流量（H8値・滞在人数推定）のポップアップを実装し、施設数・便数も可視化した。

### 相関スキャン（correlation_scan.py）

人流量を目的変数として、35変数 × 4距離（100/200/300/500m）= 計140の変数×距離の組み合わせについて、Pearson相関係数を一括計算した。

**分析対象エリア：22地点（n=22）**

計算した変数カテゴリ：
- 飲食系（飲食店・カフェ・バー・居酒屋・飲食合計・飲食密度）
- 商業系（スーパー・大規模SC・全店舗数・全店舗密度）
- 交通系（バス停数・バス総便数・バス平均便数・バス便数密度）
- 緑地系（緑地件数・緑地面積・緑地面積比率）
- 駐車場系（件数・面積・面積比率）
- 多様性指標（業種多様性H：8分類シャノンエントロピー）
- オフィス系（オフィス数・オフィス密度）

出力：`correlation_scan.html`（ヒートマップ＋TOP10一覧）、`correlation_scan.csv`

### PLATEAU建物データの整備（build_plateau_cache.py）

岡山市2024年PLATEAUデータ（815ファイル）から建物属性（用途・築年・高さ・建蔽率）をキャッシュ化するスクリプトを作成。実行後は建物系変数（建蔽率・用途混在度・建物年代多様性など）が相関スキャンに加わる。

## 3. 考察

### 3-1. 「緑地件数」vs「緑地面積」の逆転現象

最も印象的な発見は、同じ「緑地」でも**件数（小公園が多い）は正の相関、面積（大きな公園）は負の相関**という逆転現象である。

- 緑地件数200m：**r = +0.450**
- 緑地面積500m：**r = −0.317**

これはJane Jacobsの議論と一致する。Jacobsは『アメリカ大都市の死と生』で「大公園は街を分断し、人の流れを切り断つ」と指摘した。岡山の場合、旭川沿いの大きな緑地帯は人流の境界線になっている一方、路地に面した小さなポケットパークや街角の緑は人が集まる場所に付随していると考えられる。

**解釈**：面積の大きな公園は「目的地」であり日常的な通行には寄与しない。小さな緑地は商業集積地の周辺に存在しやすく、単に共変量として相関が出ている可能性もある。どちらの因果方向かは別途検討が必要。

### 3-2. バス「平均便数」と「バス停数・総便数」の逆転

- バス平均便数200m：**r = +0.386**（正）
- バス停数100m：**r = −0.232**（負）
- バス総便数100m：**r = −0.232**（負）

「バスがたくさん通る場所」が人流が多いのではなく、「1停留所あたりの便数が多い場所」が人流と正に相関している。停留所が多いだけの場所（停留所が分散している地区）は逆に人流が低い。

**解釈**：交通ハブとしての「質」が重要で、量ではない。幹線バスが集中する停留所（桃太郎大通り周辺など）が高人流と一致し、郊外への中継点的な停留所は人流が少ない傾向がある。

### 3-3. 業種多様性H は「近距離」より「広域」で効く

- H（業種多様性）500m：**r = +0.306**
- H（業種多様性）300m：r = +0.179
- H（業種多様性）100m：r = +0.090

半径が大きくなるほど相関が強くなる。これはJane Jacobsの「短いブロック・多様な用途」が街区スケール（100m）ではなく、もう少し広い生活圏（500m = 徒歩6分程度）のスケールで機能していることを示唆する。

**解釈**：「この角に何屋がある」というミクロな多様性より、「この地区を歩けばいろんな施設がある」という面的な多様性が人流を生む。

### 3-4. カフェは「結果」ではなく「静かな場所」の指標か

カフェ数が強い負の相関（r = −0.388）を示したことは直感に反する。カフェが多い場所は賑わっていないのか？

**考えられる解釈**：
1. カフェは「落ち着ける場所」を求めて人通りの少ない路地に立地しやすい（裏通りのカフェ文化）
2. 表町商店街エリアなど、シャッター化が進んでいるがカフェだけが残っているエリアが存在する
3. 人流データが「通過する人」を主に捉えており、カフェの中で長時間過ごす人は計上されにくい

この変数は解釈に要注意。カフェ＝賑わいという単純な等号は成立しない。

### 3-5. 今後の分析への示唆

現状のPEARSON相関はn=22と小さく、個々の相関係数の有意水準は参考程度（r > 0.4 でp < 0.05 程度）。今後の方向性：

1. **PLATEAU建物データの投入**：建蔽率・用途混在度・建物年代多様性を加えることで「都市形態」変数が加わり、Jane Jacobsの4条件をより直接的に検証できる。特に「古い建物」（年代多様性が高い）と人流の関係は重要な仮説。

2. **交絡変数の制御**：距離帯ごとの主要変数間でMulticollinearity（多重共線性）がある可能性が高い（緑地件数とオフィス数など）。主成分分析や重回帰への発展が必要。

3. **空間的自己相関の確認**：隣接エリア同士の人流は似ていることが多い（Moran's I検定）。空間ラグを考慮した分析が精度を上げる。

4. **カテゴリ別の分析**：22エリアを「商業中心部」「住宅混在」「交通結節点」などに分けて分析すると、それぞれで異なるドライバーが見えてくる可能性がある。
